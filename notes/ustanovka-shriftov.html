Установка шрифтов в Linux

1 Апреля 2019

В этой заметке мы рассмотрим 2 способа установки шрифтов в Linux.

Для хранения шрифтов в Linux есть две основные папки: /usr/share/fonts — для шрифтов, доступных всем пользователям, и ~/.fonts — для шрифтов, доступных только текущему пользователю. Самые популярные форматы шрифтов: TrueType Font (TTF, .ttf) и OpenType Font (OTF, .otf). Также поддерживается великое множество других форматов.

Для начала нужно скачать шрифт и, если он а архиве, разархивировать. Можно открыть файл шрифта через файловый менеджер, и откроется программа для управления шрифтами (например, gnome-font-viewer в GNOME Shell), но при условии, что таковая имеется на устройстве. Дальше можно нажать кнопку "Установить", тогда шрифт автоматически установится.

Если у вас в архиве не один файл шрифта(regular, bold, italic и т.д.), то проще установить шрифты через терминал. Перед установкой их нужно поместить в папку с общим названием, если при разархивировании это не сделалось. Для глобальной установки выполняем в терминале следующую команду:

sudo cp -r ~/Downloads/Папка_со_шрифтами /usr/share/fonts

Для установки текущему пользователю сначала нужно создать папку .fonts, если вы её ещё не создали:

mkdir ~/.fonts

Устанавливаем шрифты:

cp -r ~/Downloads/Папка_со_шрифтами ~/.fonts/

Теперь обновляем кеш шрифтов системы:

fc-cache -f -v

Конец
